# DY 选股器策略筛选指南

## 策略分类

### 💡 推荐策略（核心策略）

#### 🔥 激进策略
**筛选条件**：`buy == True`

**适用场景**：
- 追求高收益，能承受较高风险
- 短线交易者
- 有充足时间盯盘

**特点**：
- ✅ 捕捉所有买入信号
- ✅ 机会最多
- ⚠️ 假信号可能较多
- ⚠️ 需要及时止损

**使用建议**：
```python
# 激进策略
results[results['buy'] == True]

# 建议配合：
# - 设置 5-8% 止损
# - 持仓时间 1-5 天
# - 分散投资，单只不超过 10%
```

**预期收益**：高风险高收益，单次 5-15%

---

#### ✅ 稳健策略
**筛选条件**：`buy == True AND up1 == True`

**适用场景**：
- 追求稳定收益
- 中线交易者
- 风险偏好中等

**特点**：
- ✅ 买入信号 + 短期突破确认
- ✅ 成功率较高
- ✅ 风险可控
- ⚠️ 机会相对较少

**使用建议**：
```python
# 稳健策略
results[(results['buy'] == True) & (results['up1'] == True)]

# 建议配合：
# - 设置 8-10% 止损
# - 持仓时间 5-15 天
# - 单只持仓 15-20%
```

**预期收益**：中等风险中等收益，单次 8-20%

---

#### ⭐ 最强策略
**筛选条件**：`buy == True AND up3 == True`

**适用场景**：
- 追求确定性机会
- 长线交易者
- 重仓机会

**特点**：
- ✅ 买入信号 + 强势突破（蓝带突破黄带）
- ✅ 成功率最高
- ✅ 趋势最强
- ⚠️ 机会最少（精选）

**使用建议**：
```python
# 最强策略
results[(results['buy'] == True) & (results['up3'] == True)]

# 建议配合：
# - 设置 10-15% 止损
# - 持仓时间 15-30 天
# - 可重仓，单只 20-30%
```

**预期收益**：低风险高收益，单次 15-40%

---

#### ⚠️ 风险警示
**筛选条件**：`sell == True`

**适用场景**：
- 持仓监控
- 风险控制
- 减仓参考

**特点**：
- ✅ 顶背离 + MACD 死叉
- ✅ 及时发现风险
- ⚠️ 不是强制卖出信号

**使用建议**：
```python
# 风险警示
results[results['sell'] == True]

# 建议操作：
# - 持仓股票出现此信号，考虑减仓 30-50%
# - 设置移动止损保护利润
# - 观察后续走势，可能是短期调整
```

**预期作用**：避免 10-30% 的回撤

---

### 🎯 组合筛选（进阶策略）

#### 买入 + 任意上涨
**筛选条件**：`buy == True AND (up1 OR up2 OR up3)`

**说明**：买入信号 + 至少一个上涨趋势确认

**适用**：稳健策略的扩展版，机会更多

---

#### 买入 + UP1
**筛选条件**：`buy == True AND up1 == True`

**说明**：买入信号 + 突破蓝带上轨（短期阻力）

**适用**：短期交易，快进快出

---

#### 买入 + UP2
**筛选条件**：`buy == True AND up2 == True`

**说明**：买入信号 + 突破黄带上轨（长期阻力）

**适用**：中期交易，趋势较强

---

#### 买入 + UP3
**筛选条件**：`buy == True AND up3 == True`

**说明**：买入信号 + 蓝带突破黄带（强势信号）

**适用**：长期交易，最强机会（同最强策略）

---

#### 卖出 + 下跌
**筛选条件**：`sell == True AND (down1 OR down2 OR down3)`

**说明**：卖出信号 + 下跌趋势确认

**适用**：强烈风险警示，建议减仓或清仓

---

#### 强势上涨
**筛选条件**：`up2 == True AND up3 == True`

**说明**：同时突破长期阻力且蓝带强于黄带

**适用**：寻找强势股，即使没有买入信号也值得关注

---

#### 弱势下跌
**筛选条件**：`down2 == True AND down3 == True`

**说明**：同时跌破长期支撑且蓝带弱于黄带

**适用**：避免这些股票，或做空机会

## 注意事项

### 1. 策略不是万能的
- 技术指标只是参考，不保证 100% 成功
- 需要结合基本面、市场环境、个股情况

### 2. 风险控制最重要
- 无论哪种策略，都要设置止损
- 不要重仓单只股票
- 分散投资降低风险

### 3. 持续跟踪
- 每日收盘后重新筛选
- 监控持仓股票是否出现卖出信号
- 及时调整仓位

### 4. 市场环境
- 牛市：激进策略效果更好
- 震荡市：稳健策略更合适
- 熊市：最强策略 + 严格止损

### 5. 回测验证
- 建议先用历史数据回测
- 小资金实盘验证
- 找到适合自己的策略
