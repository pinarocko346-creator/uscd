## 📊 API接口

Web服务器提供8个RESTful API：

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/status` | GET | 获取系统状态 |
| `/api/holdings` | GET | 获取持仓信息 |
| `/api/run` | POST | 运行全部策略 |
| `/api/run_layer/<name>` | POST | 运行单个层级 |
| `/api/history` | GET | 获取运行历史 |
| `/api/performance` | GET | 获取性能数据 |
| `/api/config` | GET | 获取配置信息 |

## 🔒 安全建议

### 开发环境（当前配置）

```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

- ✅ 适合本地开发
- ⚠️ 允许局域网访问
- ⚠️ Debug模式开启

### 生产环境建议

1. **关闭Debug模式**
   ```python
   app.run(debug=False, host='127.0.0.1', port=5000)
   ```

2. **添加身份验证**
   - 使用Flask-Login
   - 添加用户登录系统

3. **使用HTTPS**
   - 配置SSL证书
   - 加密数据传输

4. **使用生产服务器**
   ```bash
   pip install gunicorn
   gunicorn -w 4 -b 0.0.0.0:5000 web_server:app
   ```
