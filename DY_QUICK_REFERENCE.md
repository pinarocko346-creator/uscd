## 📊 信号说明

### 买卖信号
- **BUY**：底背离 + MACD 金叉 → 买入机会
- **SELL**：顶背离 + MACD 死叉 → 卖出警示

### 趋势信号
- **UP1**：突破蓝带上轨（短期阻力）
- **UP2**：突破黄带上轨（长期阻力）
- **UP3**：蓝带突破黄带（强势信号）⭐
- **DN1**：跌破蓝带下轨（短期破位）
- **DN2**：跌破黄带下轨（长期破位）
- **DN3**：蓝带跌破黄带（弱势信号）⚠️

## 🎲 快速决策

### 我该用哪个策略？

```
保守型 → 最强策略（BUY + UP3）
  └─ 机会少但确定性高

平衡型 → 稳健策略（BUY + UP1）
  └─ 机会和风险平衡

激进型 → 激进策略（BUY）
  └─ 机会多但需要严格止损
```

### 看到信号怎么办？

```
BUY 信号
  ├─ 无趋势 → 小仓位试探（5-10%）
  ├─ + UP1  → 正常仓位（15-20%）
  ├─ + UP2  → 加大仓位（20-25%）
  └─ + UP3  → 重仓机会（25-30%）

SELL 信号
  ├─ 持仓 → 减仓 30-50%
  ├─ + DN1 → 减仓 50-70%
  └─ + DN3 → 考虑清仓
```

## 📞 常见问题

**Q: 哪个策略最好？**
A: 没有最好，只有最适合。根据你的风险偏好和交易风格选择。

**Q: 可以同时用多个策略吗？**
A: 可以！建议组合使用，分散风险。

**Q: 信号准确率多少？**
A: 没有 100% 准确的信号，重要的是风险控制和资金管理。

**Q: 多久筛选一次？**
A: 建议每日收盘后筛选一次。

**Q: 需要盯盘吗？**
A: 激进策略需要，稳健和最强策略可以不盯盘。
