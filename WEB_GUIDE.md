## 🚀 启动Web界面

### 1. 安装依赖

```bash
cd ~/three-layer-quant-system
pip install Flask
```

或者安装所有依赖：

```bash
pip install -r requirements.txt
```

### 2. 启动服务器

```bash
python web_server.py
```

你会看到：

```
================================================================================
三层量化系统 - Web界面
================================================================================

启动Web服务器...
访问地址: http://localhost:5000

按 Ctrl+C 停止服务器

================================================================================
```

### 3. 访问界面

在浏览器中打开：

```
http://localhost:5000
```

或者从其他设备访问（如果在同一网络）：

```
http://你的IP地址:5000
```

## 🎨 界面功能

### 1. 仪表板概览

- **实时时间显示**：显示当前时间和系统状态
- **资金概览**：总资金、总收益率、夏普比率、最大回撤
- **三层策略卡片**：每层的详细信息和独立运行按钮

### 2. 控制面板

- **🚀 运行全部策略**：一键运行三层策略
- **🔄 刷新数据**：手动刷新所有数据
- **单层运行**：每层都有独立的运行按钮

### 3. 收益曲线

- 使用Chart.js绘制的交互式收益曲线
- 鼠标悬停显示详细数据
- 自动更新

### 4. 持仓详情

- **基石层持仓**：显示所有持仓股票和权重
- **轮动层持仓**：显示所有持仓股票和权重
- **择时层状态**：显示当前仓位百分比

### 5. 运行日志

- 实时显示系统运行日志
- 不同类型的日志用不同颜色标识
  - 绿色：成功
  - 红色：错误
  - 蓝色：信息

4. 所有数据自动更新

- 也可以手动点击"🔄 刷新数据"按钮
- 时间显示每秒更新

## 📊 API接口

Web服务器提供以下API接口：

### 获取系统状态

```
GET /api/status
```

返回：总资金、各层仓位、持仓数量等

### 获取持仓信息

```
GET /api/holdings
```

返回：各层的详细持仓列表

### 运行全部策略

```
POST /api/run
```

Body: `{"date": "2026-03-03"}` (可选)

### 运行单个层级

```
POST /api/run_layer/<layer_name>
```

layer_name: `layer1`, `layer2`, 或 `layer3`

### 获取性能数据

```
GET /api/performance
```

返回：收益曲线数据和性能指标

### 获取配置信息

```
GET /api/config
```

返回：系统配置参数

## 🔧 自定义配置

### 修改端口

编辑 `web_server.py` 最后一行：

```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 改为其他端口
```

### 修改样式

编辑 `static/css/style.css` 自定义颜色、字体等

### 添加新功能

1. 在 `web_server.py` 添加新的API路由
2. 在 `static/js/main.js` 添加前端逻辑
3. 在 `templates/index.html` 添加UI元素

## 📱 移动端访问

1. 确保手机和电脑在同一WiFi网络
2. 查看电脑IP地址：
   ```bash
   # Mac/Linux
   ifconfig | grep "inet "

   # Windows
   ipconfig
   ```
3. 在手机浏览器访问：`http://电脑IP:5000`

## 🔒 安全提示

**重要**：当前配置允许所有IP访问（`host='0.0.0.0'`）

如果只想本地访问，修改为：

```python
app.run(debug=True, host='127.0.0.1', port=5000)
```

生产环境建议：
- 使用HTTPS
- 添加身份验证
- 使用生产级WSGI服务器（如Gunicorn）
