# DY 选股器更新说明 v1.1

## 更新内容

### 🎯 新增策略筛选功能

#### 1. 推荐策略（4种核心策略）

**🔥 激进策略**
- 筛选条件：买入信号
- 适合：短线交易，追求高收益
- 特点：机会多，需要严格止损

**✅ 稳健策略**
- 筛选条件：买入信号 + UP1（突破蓝带）
- 适合：中线交易，平衡收益风险
- 特点：成功率高，风险可控

**⭐ 最强策略**
- 筛选条件：买入信号 + UP3（蓝带突破黄带）
- 适合：长线交易，重仓机会
- 特点：成功率最高，机会最少

**⚠️ 风险警示**
- 筛选条件：卖出信号
- 适合：持仓监控，风险控制
- 特点：及时发现风险，避免损失

#### 2. 组合筛选（7种进阶策略）

- 买入 + 任意上涨
- 买入 + UP1
- 买入 + UP2
- 买入 + UP3
- 卖出 + 下跌
- 强势上涨（UP2 + UP3）
- 弱势下跌（DOWN2 + DOWN3）

#### 3. 基础筛选（5种基础筛选）

- 全部
- 买入信号
- 卖出信号
- 上涨趋势
- 下跌趋势

### 🎨 界面优化

1. **分组显示**：策略按类型分组，更清晰
2. **颜色区分**：不同策略使用不同颜色标识
3. **数量提示**：每个筛选条件显示符合的股票数量
4. **策略说明**：每个策略标签下方显示简要说明

### 📊 后端增强

1. **新增 API 筛选参数**：
   - `aggressive` - 激进策略
   - `stable` - 稳健策略
   - `strongest` - 最强策略
   - `risk_alert` - 风险警示
   - `buy_up1`, `buy_up2`, `buy_up3` - 买入组合
   - `buy_any_up` - 买入 + 任意上涨
   - `sell_down` - 卖出 + 下跌
   - `strong_up` - 强势上涨
   - `weak_down` - 弱势下跌

2. **智能统计**：自动计算每种策略的股票数量

## 使用示例

### 场景 1：寻找短线机会

1. 启动 Web 服务
2. 运行筛选
3. 点击"🔥 激进策略"
4. 查看所有买入信号的股票
5. 选择 3-5 只分散投资

### 场景 2：寻找确定性机会

1. 启动 Web 服务
2. 运行筛选
3. 点击"⭐ 最强策略"
4. 查看买入 + UP3 的股票
5. 重仓 1-2 只最强股票

### 场景 3：监控持仓风险

1. 启动 Web 服务
2. 运行筛选
3. 点击"⚠️ 风险警示"
4. 查看是否有持仓股票出现卖出信号
5. 及时减仓或止损

### 场景 4：寻找强势股

1. 启动 Web 服务
2. 运行筛选
3. 点击"强势上涨 (UP2+UP3)"
4. 查看同时满足两个上涨条件的股票
5. 关注这些强势股

## 更新后的工作流

```
1. 启动 Web 服务
   ./start_dy_web.sh

2. 访问界面
   http://localhost:5001

3. 设置参数并开始筛选
   - 最低股价：根据需求设置
   - 最低交易额：根据需求设置
   - 点击"开始筛选"

4. 查看推荐策略
   - 根据风险偏好选择策略
   - 查看每个策略的股票数量

5. 点击策略查看详情
   - 查看符合条件的股票列表
   - 分析价格和趋势信号

6. 导出结果
   - 点击"导出 CSV"保存数据
   - 进一步分析或记录

7. 持续监控
   - 每日运行筛选
   - 关注"风险警示"
   - 及时调整仓位
```
