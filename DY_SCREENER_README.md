# DY 指标美股选股器

基于 TradingView Pine Script 的 DY 指标，实现自动化美股选股系统。

## 功能特点

1. **自动遍历市场股票**：自动获取 S&P 500 和 NASDAQ 100 成分股
2. **智能过滤**：
   - 过滤价格低于 $0.1 的股票
   - 过滤日交易额小于 $10,000,000 的股票
3. **技术指标**：
   - 蓝带（EMA 23/24）和黄带（EMA 89/90）
   - MACD 背离信号
   - 买卖信号识别
4. **并发处理**：多线程并发分析，提高效率

## 安装依赖

```bash
pip install yfinance pandas numpy
```

## 使用方法

### 基本用法

```bash
# 使用默认参数运行
python dy_stock_screener.py

# 自定义筛选条件
python dy_stock_screener.py --min-price 1.0 --min-volume 50000000

# 指定股票列表
python dy_stock_screener.py --symbols AAPL MSFT GOOGL TSLA NVDA

# 调整并发线程数
python dy_stock_screener.py --workers 20

# 指定输出文件
python dy_stock_screener.py --output my_results.csv
```

### 参数说明

- `--min-price`: 最低股价（美元），默认 0.1
- `--min-volume`: 最低日交易额（美元），默认 10,000,000
- `--workers`: 并发线程数，默认 10
- `--symbols`: 指定股票代码列表（可选）
- `--output`: 输出文件名，默认 dy_screener_results.csv

## 输出说明

### 控制台输出

程序会在控制台显示：
1. 买入信号的股票列表
2. 卖出信号的股票列表
3. 各类趋势信号的统计

### CSV 文件

结果会保存到 CSV 文件，包含以下字段：
- `symbol`: 股票代码
- `price`: 当前价格
- `buy`: 买入信号（True/False）
- `sell`: 卖出信号（True/False）
- `up1`: 突破蓝带上轨
- `up2`: 突破黄带上轨
- `up3`: 蓝带突破黄带
- `down1`: 跌破蓝带下轨
- `down2`: 跌破黄带下轨
- `down3`: 蓝带跌破黄带

## 技术指标说明

### 蓝带和黄带

- **蓝带**：短期趋势带
  - 上轨：EMA(high, 24)
  - 下轨：EMA(low, 23)
- **黄带**：长期趋势带
  - 上轨：EMA(high, 89)
  - 下轨：EMA(low, 90)

### 趋势信号

- **UP1**: 收盘价突破蓝带上轨
- **UP2**: 收盘价突破黄带上轨
- **UP3**: 蓝带下轨突破黄带上轨（强势信号）
- **DOWN1**: 收盘价跌破蓝带下轨
- **DOWN2**: 收盘价跌破黄带下轨
- **DOWN3**: 蓝带下轨跌破黄带下轨（弱势信号）

### 买卖信号

- **买入信号**：底背离 + MACD DIFF 上穿 DEA
- **卖出信号**：顶背离 + MACD DIFF 下穿 DEA

## 注意事项

1. **数据源**：使用 yfinance 获取数据，需要网络连接
2. **运行时间**：分析大量股票需要时间，建议使用合理的并发线程数
3. **数据延迟**：免费数据源可能有延迟，不适合高频交易
4. **风险提示**：本工具仅供参考，不构成投资建议

## 与 Pine Script 的对应关系

| Pine Script | Python 实现 |
|------------|------------|
| `ta.ema()` | `pandas.ewm().mean()` |
| `ta.barssince()` | 自定义 `bars_since()` 函数 |
| `ta.crossover()` | 比较前后两个值的大小关系 |
| `request.security()` | `yfinance.download()` |
| 硬编码 40 只股票 | 自动获取市场股票列表 |
| TradingView 表格显示 | CSV 文件 + 控制台输出 |
