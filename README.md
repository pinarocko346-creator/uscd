# 三层量化交易系统

一个基于多因子、机器学习和市场择时的完整量化交易系统。

## 🎯 系统架构

### 三层策略设计

```
┌─────────────────────────────────────────────────────────┐
│                    三层量化系统                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  第一层：基石层（50%仓位）                               │
│  ├─ 标的：沪深300成分股                                  │
│  ├─ 策略：多因子回归选股                                 │
│  ├─ 调仓：每月第一个交易日                               │
│  └─ 持仓：20只股票 × 2.5%                               │
│                                                          │
│  第二层：轮动层（30%仓位）                               │
│  ├─ 标的：中证500成分股                                  │
│  ├─ 策略：随机森林动态因子                               │
│  ├─ 调仓：每周一                                         │
│  └─ 持仓：15只股票 × 2%                                 │
│                                                          │
│  第三层：择时层（20%仓位）                               │
│  ├─ 标的：沪深300ETF或现金                               │
│  ├─ 策略：KNN+SVM+NLP三模型投票                          │
│  ├─ 调仓：每日                                           │
│  └─ 持仓：全进/全出/做空                                 │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 📊 策略详解

### 第一层：基石层（多因子回归）

**目标**：稳定收益，长期持有优质股票

**因子体系**：
- 价值因子：PE、PB（权重25%+25%）
- 质量因子：ROE、毛利率（权重20%+15%）
- 动量因子：20日涨跌幅（权重15%）

**执行流程**：
1. 每月第一个交易日09:30执行
2. 从沪深300中筛选20只股票
3. 因子标准化后加权评分
4. 等权重配置（每只2.5%）

**风险控制**：
- 单股止损：-15%
- 组合回撤：-20%
- 连续跑输基准：2个月

### 第二层：轮动层（随机森林）

**目标**：捕捉中短期机会，动态调整

**策略逻辑**：
1. 每周五计算8个候选因子的IC值
2. 选出IC值最高的5个"黄金因子"
3. 用随机森林预测未来5日收益
4. 选出预测收益最高的15只股票

**执行流程**：
1. 周五收盘后计算
2. 周一09:30执行调仓
3. 每只股票2%仓位

**风险控制**：
- 模型过拟合检测（R²>0.85）
- 黄金因子剧变预警
- 高换手率限制（<66%）
- 连续亏损暂停（3周）

### 第三层：择时层（三模型投票）

**目标**：规避系统性风险，提升整体收益

**三个模型**：

1. **KNN模型**（历史相似度）
   - 找出历史上最相似的5天
   - 统计次日涨跌概率

2. **SVM模型**（技术指标分类）
   - 输入：RSI、MACD、布林带等
   - 输出：涨/跌/震荡

3. **NLP模型**（市场情绪）
   - 抓取雪球、东方财富评论
   - 情绪分析：乐观/悲观/中性

**投票机制**：
- 3票看多：满仓ETF（20%）
- 2票看多：半仓ETF（10%）
- 1票看多：观望（0%）
- 0票看多：做空或空仓

## 🚀 快速开始

### 1. 安装依赖

```bash
cd ~/three-layer-quant-system
pip install -r requirements.txt
```

### 2. 配置参数

编辑 `config/system_config.py`：
```python
# 资金配置
TOTAL_CAPITAL = 1000000  # 总资金

# 仓位配置
LAYER1_WEIGHT = 0.50  # 基石层50%
LAYER2_WEIGHT = 0.30  # 轮动层30%
LAYER3_WEIGHT = 0.20  # 择时层20%

# 数据源
DATA_SOURCE = "tushare"  # 或 "akshare"
```

### 3. 运行系统

```bash
# 完整运行（三层同时）
python main.py

# 单独运行某一层
python layer1_cornerstone/main.py  # 基石层
python layer2_rotation/main.py     # 轮动层
python layer3_timing/main.py       # 择时层
```

### 4. 回测

```bash
# 回测指定时间段
python backtest.py --start 2020-01-01 --end 2025-12-31

# 查看回测报告
python analysis/generate_report.py
```

## 🔧 配置说明

### 因子配置

在 `config/factor_config.py` 中可以调整：
- 因子权重
- 因子计算方法
- 因子筛选阈值

### 风险配置

在 `config/risk_config.py` 中可以调整：
- 止损比例
- 回撤阈值
- 仓位限制

### 数据源配置

支持多种数据源：
- Tushare（推荐）
- AKShare
- Wind（需要付费）
- 自定义数据源

## 🎓 使用建议

### 1. 初始阶段

- 先用小资金测试（10-20万）
- 观察3个月以上
- 记录每次调仓原因

### 2. 参数调优

- 根据回测结果调整因子权重
- 优化模型参数
- 调整仓位配置

### 3. 风险管理

- 严格执行止损
- 定期检查持仓
- 关注市场环境变化

## ⚠️ 风险提示

1. **历史表现不代表未来收益**
2. **模型可能失效**：市场环境变化时需要调整
3. **交易成本**：频繁调仓会增加成本
4. **滑点影响**：实盘执行可能与回测有差异
5. **数据质量**：依赖数据源的准确性
